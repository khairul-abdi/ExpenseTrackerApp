# [ExpenseTrackerApp](https://mernexpensetracker.herokuapp.com/)

### In this App, we need 5 dependencies below:

- Express.js
- React.js
- Axios
- MongoDB
- concurrently
- dotenv  
- morgan
- colors


### Usage
```
change config.env file in config folder
```

``` 
 npm install
 cd client npm install
 cd ..
 
 # Run front and backend
 npm run dev
 
 # Backend only
 npm run server
 
 # Frontend only
 npm run client
 
 # Build client
 cd client
 npm run build
 
 # Production
 npm start

```
